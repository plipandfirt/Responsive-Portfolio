About Me

Laurie Anderson is a Project Manager, with 20 years of experience inspiring teams to reach for excellence in their work and within themselves.

In 2018, she stepped out of her comfort zone to explore the world Full Stack Development. Through a Coding Boot Camp offered by UNH's Professional Development & Training she now understands HTML5, CSS3, JavaScript, jQuery, Node.js, Express.js, React.js, Database Theory, MongoDB, MySQL, Command Line, Git and more.

In four months she's learned so much:

  • Front-End Web Development
  • HTML 5 Web Development
  • Node.Js Coding
  • JavaScript Coding
  • MERN Stack Coding
  • Java Back-End Coding
  • Object-Oriented Coding

Laurie lives in New Hampshire with several four-legged creatures. When she's not coding she retreats to the woods, forever searching for the perfect fishing hole.
